# anon V6 - WhatsApp Assistant Bot

A smart WhatsApp assistant bot for pairing and managing devices.

## Features

- 🔗 Easy WhatsApp device pairing
- 📱 Web interface for mobile & desktop
- 🟢 Real-time connection status
- ⚡ Fast and lightweight

## Installation

### Termux (Android)

```bash
# Update packages
pkg update && pkg upgrade -y

# Install required packages
pkg install nodejs python git -y

# Install Python dependencies
pip install flask

# Clone repository
git clone https://github.com/bothost-source/loner-bantool.git

# Enter directory
cd anon-v6

# Install Node.js dependencies
npm install

# Start the bot
bash start.sh
